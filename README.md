# Shrey16 Claude Code Plugins

A marketplace of Claude Code plugins.

## Installation

```bash
/plugin marketplace add shrey16/claude-plugins
```

## Available Plugins

| Plugin | Description | Version |
|--------|-------------|---------|
| [pm-copilot](https://github.com/shrey16/pm-copilot) | Autonomous product management copilot — feature drilling, spec generation, checkpoint-based implementation, and product health tracking | 0.1.0 |

## Installing a Plugin

After adding the marketplace:

```bash
/plugin install pm-copilot@shrey16-plugins
```
